# customer-list-search

I would like to have a list of customers, where each customer contains fields like: name, surname, phone, email. There should be a search field to filter by any property. There should be a button to add new customer, which shows a form.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/customer-list-search.git
cd customer-list-search
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
